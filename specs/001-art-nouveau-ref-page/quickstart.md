# Quickstart: Art Nouveau Reference Page

**Feature**: Art Nouveau Reference Page
**Date**: 2026-02-09

## How to View

1.  **Locate the File**:
    - `inspiration/art_nouveau_reference.html`

2.  **Open in Browser**:
    - Right-click the file in your file explorer and choose "Open With > Chrome/Firefox/Safari".
    - Or drag and drop the file into a new browser tab.
    - Or serve via a simple HTTP server (optional):
      ```bash
      # From project root
      python3 -m http.server
      # Open http://localhost:8000/inspiration/art_nouveau_reference.html
      ```

3.  **Inspect**:
    - Use Developer Tools (F12) to verify the CSS classes and structure.
    - Resize the window to test responsiveness (Mobile First: start at 320px).

## Troubleshooting

- **Fonts Not Loading**: Ensure you have an active internet connection (Google Fonts CDN requirement).
- **Styles Missing**: Ensure you have an active internet connection (Tailwind CDN requirement).
- **Dark Mode**: Check your OS preference or toggle the `dark` class on the `<html>` element manually (no JS toggle implemented).
